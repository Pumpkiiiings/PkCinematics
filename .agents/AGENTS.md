## PacketEvents

- **Dependencia Gradle**:
  ```groovy
  implementation 'com.github.retrooper:packetevents-spigot:2.13.0'
  ```
- **Javadocs**: https://javadocs.packetevents.com/spigot/

Nunca usar versiones antiguas como 2.5.0, 2.3.0, etc.
Antes de usar cualquier método de PacketEvents, consultar los javadocs para verificar el API correcto.
